## How to run this Project?

<p>
1. Clone this project:

```
git clone https://github.com/floatingnahid22/Rest-Countries-Api-with-Angular.git
```

<br>
2. Open this project on VS Code and go to project directory then make

```
npm install
```

<br>
3. To run Project

```
ng serve
```

</p>
